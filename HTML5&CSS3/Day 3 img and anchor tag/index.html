Image Tag

1. src Attribute : src Attribute are used to provide source or path of the image.
2. height and width Attribute : The width attribute are used to give width of an image, in pixels. The Height attribute are used to give Height of an image, in pixels.
3. alt Attribute : alt tag are used to provide alternative text of the image, if the image is cannot displayed for some reasons or we can provide wrong source of the image.
Local Image You can add a local image to your HTML page and then the first download or save the image in the same folder and then provide the source of the image. Ex: src= "imageName.png"
